## 概要

このプログラムはBox.comのPDFファイルをダウンロードするためのものです。


## 使い方

下記のコマンドを実行してください。

```
git clone https://github.com/101ta28/box_pdf_downloader.git
cd box_pdf_downloader
pip install -r requirements.txt
python main.py
```
